## Q & A

**1. TCP 和 UDP 区别？**

* TCP 提供面向连接、可靠的字节流服务；UDP 提供无连接、不可靠的数据报服务
* 对系统资源的要求(TCP 较多，UDP 少)
* TCP 保证数据正确性，数据顺序；UDP 不能保证
* TCP 有超时重发、流量控制等机制，而 UDP 没有
